报错The bean ‘item-service.FeignClientSpecification’ could not be registered. A bean with that name

2020-05-19- 2020-06-02
解决错误 > Java > Bean-JavaERRORspringbootbean

2020-04-16 15:27:55.970 ERROR 14704 — [ main] o.s.b.d.LoggingFailureAnalysisReporter : 

*****************************
APPLICATION FAILED TO START
*****************************
Description:
The bean 'item-service.FeignClientSpecification' could not be registered. A bean with that name has already been defined and overriding is disabled.

翻译：Bean’item-service.FeignClientSpecification’无法注册。已经定义了具有该名称的Bean，并且禁用了覆盖。

解决方法：启用，在配置文件yml中添加

spring:
  main:
    allow-bean-definition-overriding: true
